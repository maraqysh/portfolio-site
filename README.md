# Сайт-визитка Кирилла

Одностраничное портфолио веб-разработчика. Сайт работает без сборки и зависимостей.

## Как открыть

Откройте `index.html` двойным кликом или запустите локальный сервер в этой папке:

```bash
python3 -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.

## Что заменить перед публикацией

- В `index.html` замените названия, описания и ссылки в блоке проектов.
- Замените `hello@example.com` на рабочую почту.
- Замените `https://t.me/username` на ссылку на свой Telegram.
- При необходимости измените город в строке «Веб-разработчик · Москва».

Все основные цвета находятся в начале `styles.css`.
